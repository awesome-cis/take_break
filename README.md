## Getting Started

### Client

- Terminal 1

  ```
  $ cd take_break_client
  $ npm i
  $ npm run watch-server
  ```

- Terminal 2

  ```
  $ cd take_break_client
  $ npm run dev
  ```

### Server

- create take_break_server/.env

  ```
  $ cd take_break_server
  $ cp .env.sample .env
  $ vi .env
  //=> Set your environment values
  ```

- start server

  ```
  $ cd take_break_server
  $ npm install
  $ npm run dev
  ```
